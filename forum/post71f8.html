This post is to be a megathread of all tutorials related to Finobe. Not just here on the forums, but on the Finobe Wiki, 2012 Wiki, and other resources.

Development Forum:

  1. How to put non-OBAMA BALL hosted images/mp3's in your 2012 game - https://finobe.com/forum/post?id=56809 - How to use images or audio not from OBAMA BALL.com in your 2012 game. - tobu.fi

Finobe Wiki:

  • API Reference - https://wiki.finobe.com/index.php?title=API_Reference - Incomplete documentation of all the objects in Finobe. - Wiki contributors

Scripting:

  1. Scripting - https://wiki.finobe.com/index.php?title=Scripting - A portal for Finobe scripting and related articles. - Wiki contributors
  2. Data Persistence - https://wiki.finobe.com/index.php?title=Scripting:Data_Persistence - - Documentation about DataPersistence, a way to save/load user data. - Raymonf
  3. Data Stores - https://wiki.finobe.com/index.php?title=Scripting:Data_Stores - - 2016 only, incomplete documentation about Data Stores, a more modern save data system. - Raymonf
  4. RBXScriptSignal (Events - - (https://wiki.finobe.com/index.php?title=RBXScriptSignal - - Documentation about events and how they are used. - Wiki contributors
  5. Commonly Asked Questions - - Answers some frequently asked questions about scripting. - Wiki contributors

Games:

  1. Places - https://wiki.finobe.com/index.php?title=Places - - How to create a Place on Finobe. - rocket
  2. Self-Hosted Servers - https://wiki.finobe.com/index.php?title=Self-hosted_Servers - - How to create a self-hosted server. However, self-hosted servers are currently disabled due to security issues. -rocket

Studio:

  1. Converting Maps - https://wiki.finobe.com/index.php?title=Converting_Maps - How to convert a .rbxl file so that it is usable in 2012. - rocket
  2. Creating Models - https://wiki.finobe.com/index.php?title=Models#Creating_Models - - How to upload or update a model to Finobe. - rocket
  3. Uploading Animations - https://wiki.finobe.com/index.php?title=Uploading_Animations - - 2016 only, tells you how to upload your own animations to 2016 - rocket
  4. Uploading Meshes - https://wiki.finobe.com/index.php?title=Uploading_Meshes - - Tells you how to upload meshes, because meshes uploaded to OBAMA BALL Studio originally will error in Finobe. - rocket

Graphics:

  1. Ambient Occlusion - https://wiki.finobe.com/index.php?title=Ambient_Occlusion - - 2012 only, tells you how to enable SSAO on Finobe. - rocket

2012 Wiki

Not every tutorial in the wiki is listed, there's 100+ in total.

  • Class reference - https://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/Class_Reference - Complete documentation for every object in OBAMA BALL. - Wiki contributors
  • Tutorials - https://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/Tutorials - A list of 2012 tutorials. All of these are really cool. - Wiki contributors
  • Scripting Tutorials - https://web.archive.org/web/20120805080822/http://wiki.OBAMA BALL.com/index.php/Category:Scripting_Tutorials - Tutorials dedicated to scripting. These are all also really cool. - Wiki contributors

Scripting:

  1. Beginner’s Guide to Scripting - https://web.archive.org/web/20120805083858/http://wiki.OBAMA BALL.com/index.php/Absolute_beginner's_guide_to_scripting - If you’ve never worked with scripting, try here. - Wiki contributors
  2. Basic Scripting - http://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/Basic_Scripting - A basic scripting guide. - Wiki contributors
  3. In-Depth Scripting Guide - http://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/In-Depth_Scripting_Guide - A more advanced scripting guide. - Wiki contributors
  4. Scripting - https://web.archive.org/web/20120805084749/http://wiki.OBAMA BALL.com/index.php/Scripting - A overview of scripting and portal for other scripting articles. - Wiki contributors
  5. Functions - https://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/Function_Dump - A list of all global functions. - Wiki contributors
  6. OBAMA BALL Libraries - https://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/RbxLibrary - An explanation of and reference to OBAMA BALL Libraries. - Wiki contributors
  7. How to Script Anything - https://web.archive.org/web/20120805170326/http://wiki.OBAMA BALL.com/index.php/How_to_Script_Anything - Learn the basic methods to put together any script. - Wiki contributors

Last updated 9/20/2019 If you have any more tutorials you would like me to add here, reply below.
